# learn-zig

## QuickStart:

> install:

```ruby

# install zig:
task install:zig

# macos + m1 cpu:
task install:zigmod:m1
# or macos + intel cpu
task install:zigmod:intel


```

> test run:

```ruby
task basic:run
```

- [basic](./packages/basic/): basic zig example
